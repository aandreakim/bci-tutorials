This is one of my first independent projects learning how to work with neural data — exploring EEG signals, understanding how to process them, and ultimately building intuition for BCI work.

This repo is based on the BCI tutorial created by [@syncrograph](https://github.com/syncrograph). The original repo can be found [here](https://github.com/syncrograph/bci-tutorials). Full credit to them for designing the tutorials and making this material accessible! 

# Setup

## 1. Create and activate a virtual environment
```bash
python -m venv venv
source venv/bin/activate   # macOS/Linux
```

## 2. Install dependencies
```bash
pip install matplotlib numpy pandas mne 'moabb[all]'
```
